## 故事簡介

1989 年，愛荷華州。農夫 Ray Kinsella 在玉米田裡聽見了一個神秘的聲音：

> *"If you build it, he will come."*

為了回應這個聲音，他犧牲了農地，在田中間蓋起一座棒球場。  
接著，第二個聲音來了，然後是第三個。

每一個聲音，都帶他更靠近一個他從未說出口的遺憾。

這是一個關於**信念、犧牲與和解**的故事。

---

## 遊戲特色

**選擇影響一切**  
每個決定都會影響你的存款、花費的天數，以及後續能做的選擇。沒有絕對的對錯，只有你認為值得的代價。

**三場關鍵打席**  
故事的三個轉折點，你需要親自拿起球棒。移動滑鼠瞄準來球，在對的時機點擊揮棒——打出安打，故事往更圓滿的方向走；揮空，也有揮空的結局。

**三種不同結局**  
故事的最後，根據你在三場打席的表現，會走向截然不同的結局：
- 🌟 **燈火長明** — 三場全勝，一切圓滿
- ⚾ **走完這段距離** — 有得有失，但走完了這段路
- 💙 **光不因你而滅** — 三場全敗，但有些事情的發生，不需要你完美

**分數與最高紀錄**  
遊戲結束時，剩餘存款與花費天數會換算成最終分數。每次遊玩都會記錄下來，挑戰自己的最高分。

**成就系統**  
8 個成就，記錄你的選擇風格與旅程。有些成就需要特定的選擇路線，有些則是……意外的驚喜。

## 操作方式

**對話推進**
- 滑鼠左鍵 / `Space` / `Enter` — 推進對話或跳過打字效果
- 點擊選項按鈕 — 做出選擇

**打擊小遊戲**
- 移動滑鼠 — 瞄準飛來的球
- 滑鼠左鍵 — 揮棒
- 準心變綠時，代表距離球夠近，可以出棒

> 好球帶內的球要打，好球帶外的球也可以揮——但要有心理準備。

## 存檔說明

遊戲會**自動存檔**，關閉頁面後再回來，可以從「繼續遊戲」接著玩。  
完成一局後存檔自動清除，下次開始新的旅程。  
歷史紀錄與最高分會永久保留，可從標題畫面的「歷史紀錄」查看。
